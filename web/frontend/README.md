# Flaky Web Frontend

React + TypeScript + Vite frontend for the Flaky quiz application.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

The frontend will be available at http://localhost:5173

## Features

- PDF quiz upload and solving
- Multiple view modes (Detailed, Compact, Grid)
- Collapsible PDF viewer
- Real-time answer display

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS Modules
